# Is an object a URL?

Is an object a URL?

- `is_esri_url()`: Is an object an ArcGIS MapServer or FeatureServer
  URL?

&nbsp;

- `is_gsheet_url()`: Is an object a Google Sheets URL?

&nbsp;

- `is_gist_url()`: Is an object a URL for a GitHub Gist?

&nbsp;

- `is_gmap_url()`: Is an object a Google Maps URL?

## Usage

``` r
is_url(x)

is_esri_url(x)

is_gsheet_url(x)

is_gist_url(x)

is_gmap_url(x)
```

## Arguments

- x:

  A object to be tested.
