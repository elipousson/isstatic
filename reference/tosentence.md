# Convert to a common sentence case

Convert to a common sentence case

## Usage

``` r
tosentence(x)
```

## Source

[Statistics
Globe](https://statisticsglobe.com/r-capitalize-first-letter-of-character-string-containing-multiple-words)

## Arguments

- x:

  a character vector, or an object that can be coerced to character by
  [`as.character()`](https://rdrr.io/r/base/character.html).
