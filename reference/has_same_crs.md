# 

- `has_same_crs()`: Do two sf, sfc, or bbox objects use the same
  coordinate reference system?

- `has_same_crs()`: Do two sf, sfc, or bbox objects use the same
  coordinate reference system?

## Usage

``` r
has_same_crs(x, y, ...)
```

## Arguments

- x, y:

  sf, sfc, or bbox objects to be compared.

- ...:

  Additional parameters passed to
  [`identical()`](https://rdrr.io/r/base/identical.html) by
  `has_same_crs()`.
